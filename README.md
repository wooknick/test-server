# Test Server

Express Test Server for Studying.

## How To Use( need to update )

1. Create OpenSSL

   ```terminal
   $> openssl req -x509 -newkey rsa:4096 -keyout key.pem -out cert.pem -days 365
   ```

2. Set Server IP

3. Run Server

```terminal
$> yarn server
$> yarn front
```

or

```terminal
$> npm run server
$> npm run front
```
